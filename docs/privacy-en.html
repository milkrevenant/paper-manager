Privacy Policy

Last updated: January 2024

Summary: Paper Manager is a local desktop application that does not collect or transmit personal information to external servers. All data is stored only on your computer.

1. Overview

Paper Manager ("the App") is a desktop application for managing academic papers. This Privacy Policy explains how the App handles data and describes optional features (Google Drive integration, AI analysis).

2. Information We Collect

2.1 Locally Stored Data (Not Collected)

The following data is stored only on your computer and is never transmitted to developers or third parties:

  • Paper PDF files and metadata (title, authors, publication info, etc.)
  • Highlights, annotations, and notes
  • Folder structure and tags
  • App settings and preferences

2.2 Google Drive Integration (Optional)

When you enable Google Drive sync:

Permission Purpose
Google Drive (read-only) List and download PDF files from selected folders
Basic profile information Display connected account (email address)

Note: Google account authentication tokens are stored only on your local computer and are never transmitted to our servers.

2.3 AI Analysis Feature (Optional)

When you use the AI paper analysis feature:

  • Paper text is sent to Google Gemini API for analysis
  • You provide your own API key, which is stored locally
  • Transmitted data is processed according to Google's Privacy Policy

3. Data Storage Location

All app data is stored locally at:

  • macOS: ~/Library/Application Support/com.papermanager.desktop/
  • Windows: %APPDATA%\com.papermanager.desktop\

4. Data Sharing

Paper Manager does not share user data with third parties. However, the App communicates with external services in the following cases:

  • Google Drive API: When sync feature is enabled
  • Google Gemini API: When AI analysis feature is used
  • Semantic Scholar API: When paper search feature is used (only search queries are transmitted)

5. Your Rights

You can at any time:

  • Delete all data by removing the app data folder
  • Disconnect your Google account
  • Disable AI analysis features

6. Security

The App implements the following security measures:

  • Secure Google authentication via OAuth 2.0
  • Local encrypted storage of API keys
  • HTTPS for all external communications

7. Children's Privacy

Paper Manager is not intended for children under 13 years of age, and we do not knowingly collect personal information from children.

8. Changes to This Policy

This Privacy Policy may be updated as needed. Significant changes will be announced through the App or the GitHub repository.
